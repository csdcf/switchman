***ROM Update Required!***

BootROM requirement for K.15.03.xxxx!

All switches must be running a BootROM version of
K.12.17 or newer in order to load K.15.03.xxxx software.

ProCurve 3500yl, 5400zl, 6200yl, and 8212zl switches
will need to be updated and running software version 
K.13.58 (BootROM K.12.17) or newer software before 
successfully loading K.15.03.xxxx software versions. 

If your switch is running a software version earlier than 
K.15, your BootROM will be updated when you upload 
K.15.03.xxxx software to your switch. 

**Please do not interrupt power to the switch during the update!!**

***Configuration Compatibility Caution***

Please follow "Best Practices for Major Software Updates" 
in the release notes for configuration compatibility.
